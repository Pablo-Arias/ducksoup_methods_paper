
# Introduction

In this repository you will find the analyses scripts of the article "DuckSoup: An open-source platform to control participants’ facial and vocal signals in real-time during social interactions", by  Pablo Arias Sarah (1,2,3,+), Guillaume Denis(2), Philippe G. Schyns (3), Rachael Jack (3), Petter Johansson (1), Lars Hall (1) and Jean-Julien Aucouturier (5)

1 Lund University Cognitive Science, Lund University, Lund, Sweden
2 STMS Lab, UMR 9912 (IRCAM/CNRS/SU), Paris, France
3 School of Neuroscience and Psychology, Glasgow University, Glasgow, United Kingdom
4 
5 FEMTO-ST Institute (CNRS/Université de Bourgogne Franche Comté), Besançon, France
+Corresponding author


In this article we show that aligning the smiles of meet up participants with smile transformation filters causally increases liking. In this repository you will find the python and R scripts to replicate all the analyses in both the main paper and the Supplementary Information. 

## Importing data
To use these scripts please download the raw data online here : XXXX

    Put the downloaded data in a folder called "data/" in the root of this project. Then execute the scripts.

## Analyses
These analyses include:
- Manipiulation Check
- Behavior analyses
- Facial mimicry anlayses
- Synchrony analyses
- Control analyses
- semantic analysis
- Heart rate analysis
- Voice anlayses

Please refer to the paper's method sections and Supplementary Information for a precise description of the methods implemented in these analyses.
